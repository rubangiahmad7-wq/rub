# rub

Website game catur sederhana.

## Cara menjalankan

1. Buka `index.html` di browser.
2. Klik bidak untuk memilih.
3. Klik petak tujuan untuk memindahkan bidak.
4. Klik tombol "Mulai Game Baru" untuk reset.

## Isi proyek

- `index.html` — tampilan papan catur dan kontrol.
- `style.css` — gaya tampilan papan dan sidebar.
- `script.js` — logika permainan, giliran, dan langkah sah.
